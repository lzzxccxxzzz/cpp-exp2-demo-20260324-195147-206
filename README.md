# 实验二 栈的应用

## 实验要求

1. 请使用顺序栈，实现任意 10 进制数的 `m (0 < m < 10)` 进制输出。
   要求：输入为十进制整数 `n`，输出为 `m` 进制字符串。
2. 使用顺序栈，实现判断表达式括号是否匹配的算法（括号为两种类型）。

注意：实验报告模板和程序代码独立，最终压缩打包提交。

## 当前仓库内容

- `main.cpp`：实验主程序，包含两个功能。
- `实验报告.md`：可直接提交或粘贴到模板中的实验报告内容。
- `实验报告模板.doc`：原始模板文件。

## 功能说明

程序提供菜单式交互：

1. 十进制数转换为 `m` 进制
2. 判断表达式中 `()` 和 `[]` 是否匹配

## 编译与运行

如本机已安装 `g++`，可使用：

```bash
g++ -std=c++17 -O2 -Wall -Wextra -pedantic main.cpp -o exp2
./exp2
```

当前仓库还提供了适配本机 MinGW 环境的批处理脚本：

```bat
build.bat
run_exp2.bat
```

如使用 Visual C++ 编译器，可使用：

```bat
cl /EHsc /std:c++17 main.cpp
main.exe
```
